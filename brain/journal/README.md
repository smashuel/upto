# Journal

Dated scratch for in-progress threads. Temporary — clean out regularly.

## Convention

- One file per thread, named `YYYY-MM-DD-slug.md`
- Frontmatter: `type: journal`, `status: in-progress | done`
- Use for: partial debugging notes, "what I tried", open questions mid-work
- When the thread completes:
  - If the conclusion matters → move the "why" to [decisions/](../decisions/) as an ADR
  - Otherwise → delete the journal entry

## Not here

- Shipped features → [features/](../features/)
- Phased work plans → [plans/](../plans/)
- Always-true project state → [project/](../project/)
- Long-lived research → [research/](../research/)
